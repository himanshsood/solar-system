## Installation
```sh
# Clone the repository
git clone https://github.com/himanshsood/solar-system.git

# Navigate to the project directory
cd client
cd backend

# Install dependencies
npm install
```

## Usage
```sh
# Start the project
npm start || nodemon index.js
```
